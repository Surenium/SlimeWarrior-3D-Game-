# SlimeWarrior-3D-Game-

## Goal

Collect All the Coins to Win

## Highlighted Features

- Sword Attack
- Health Bar
- Enemy Health Bar
- Enemy Attack
- Coin Counter
- Custom Win, Lose, Pause and Start Menu
